1. Командой npm i -D typescript установить компилятор TypeScript.
2. Проверить версию компилятора командой npx tsc -v.
3. Создать конфигурационный файл tsconfig.json, выполнив команду npx tsc --init.
4. В параметрах include и compilerOptions.rootDir указать, где будут расположены исходные файлы с кодом на TypeScript.
5. В параметре compilerOptions.outDir указать, куда компилятору складывать итоговые файлы с кодом на JavaScript.
6. При необходимости указать другие параметры компиляции.
7. Выполнить компиляцию командой npx tsc.
